Operations Dashboard
Description:

Demo dashboard for Monitoring and profiling of service endpoints.
Each endpoint can have several versions and based on the query return of each endpoint version view will be generated.

Function:

Support for viewing Latency, Uptime, Bad performing services, etc.
Easy visualization through widgets and charts.

Design:

Built with Angular 9, Angular Material and Chartjs.
Fully Responsive with PWA support.

Dummy Endpoints used from:
https://reqres.in
